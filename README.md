# Real-Time Collaborative Circle Drawing App

A real-time collaborative drawing application that allows users to draw red circles on a canvas. Built using **p5.js**, **Node.js**, and **Socket.IO**, this app enables multiple users to draw and see each other's circles in real-time.

## Features:

- Real-time collaborative drawing: All users can draw and see each other's circles instantly.
- Built using **p5.js** for drawing, **Socket.IO** for real-time communication, and **Express.js** for serving the app.

## Technologies Used:

- **Node.js**: JavaScript runtime for the server-side.
- **Express.js**: Web framework to handle HTTP requests and serve static files.
- **Socket.IO**: Real-time bidirectional event-based communication between clients and the server.
- **p5.js**: JavaScript library for creative coding, used to create the canvas and draw circles.
